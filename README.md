# New-Cointab-Assignment




## Tech Stack

**Client:** React, Redux, Chakra UI, 

**Server:** Node, Express, jwt, bycrpt.


## Run Locally

Clone the project

```bash
  https://github.com/Pradeep-PSY/New-Cointab-Assignment.git
```

Go to the project directory

```bash
  cd Frontend
  cd cointab-front
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
